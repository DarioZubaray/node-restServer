## Node Rest Server - Demo

### Peticiones:
  - {{url}}/usuario?desde=0&hasta=10 _GET usuario_
  - {{url}}/usuario _POST usuario_
  - {{url}}/usuario/:id _PUT usuario_
  - {{url}}/usuario/:id _DELETE usuario_
  - {{url}}/login _POST login_

  - / _index_

```
npm install
```
